Сайт представляет собой социальную сеть блогеров✍ Здесь можно публиковать интересные посты про путешествия и туризм⛺ Приглашаю вас к участию в качестве авторов новых публикаций!😉

После регистрации и авторизации Вы сможете публиковать свои посты и фото в любом количестве (эту фичу сделал по собственному желанию, т.к. по учебному ТЗ это не требовалось), комментировать уже существующие посты, подписываться на новые публикации понравившихся авторов, обновить пароль авторизации при необходимости.

Технологии проекта


Python

Высокоуровневый язык программирования общего назначения с динамической строгой типизацией и автоматическим управлением памятью, ориентированный на

повышение производительности разработчика, читаемости кода и его качества, а также на обеспечение переносимости написанных на нём программ.


Django

Фреймворк Python для веб-разработки, который позволяет быстро создавать масштабируемые и безопасные приложения.


Git

Система контроля версий, используемая для отслеживания изменений в коде проекта и совместной работы над ним.


CSS

Язык стилей, используемый для оформления веб-страниц и приложений.


HTML

Язык разметки для создания веб-страниц и приложений.
