Технологии проекта

Технологии, которые были применены при разработке:

Python

Высокоуровневый язык программирования общего назначения с динамической строгой типизацией и автоматическим управлением памятью,
ориентированный на повышение производительности разработчика, читаемости кода и его качества, а также на обеспечение переносимости написанных на нём программ.

Django

Фреймворк Python для веб-разработки, который позволяет быстро создавать масштабируемые и безопасные приложения.

HTML

Язык разметки для создания веб-страниц и приложений.

CSS

Язык стилей, используемый для оформления веб-страниц и приложений.

Git

Система контроля версий, используемая для отслеживания изменений в коде проекта и совместной работы над ним.
